
# Chat App

## Description
Chat App is a full-stack web application designed for real-time communication. It allows users to send and receive messages instantly through a sleek and user-friendly interface. The application is built with a focus on simplicity, performance, and scalability.

## Features

- Real-time messaging
- User authentication and authorization
- Responsive design for mobile and desktop
- Secure communication using HTTPS. and WebSockets
- Easy-to-use interface

## Installation
To set up the project locally, follow these steps:
### Backend
1. Clone the repository:
   ``hshs
   git clone https://github.com/tal6203/Chat-App.git
   cd Chat-App/BackEnd
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
33. Set up the environment variables:
   Create a .env file in the BackEnd directory and add the following:
   ```
   PORT=5000
   MONGODB_URI=your_mongodb_uri
   JWT_SECRET=your_jwt_secret
   ```
4. Start the backend server:
   ``hshs
   npm start
   ```

### Frontend
1. Navigate to the frontend directory:
   ```sh
   cd ../FrontEnd
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
33. Start the frontend development server:
   ``hshs
   npm start
   ```

## Usage
After setting up the project, you can access the chat application by navigating to `http://localhost:3000` in your web browser. Register a new account or log in with an existing account to start chatting.
